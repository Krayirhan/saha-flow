# Tasarım Referans Kuralları

İşAkış'ın UI geliştirmesinde aşağıdaki 4 referans site kullanılır.
Yeni bir sayfa, bileşen veya animasyon tasarlanırken önce bu sitelerden ilham alınmalı, sonra Tailwind CSS + shadcn/ui ile implement edilmelidir.

## Referans Siteler

1. **VibeUI** → https://vibeui.online
   - Dashboard layoutları, kart bileşenleri, genel SaaS panel UI'ı

2. **Refero Design Styles** → https://styles.refero.design
   - Gerçek ürünlerden alınmış stil desenleri, typography, spacing referansı

3. **Aceternity UI** → https://ui.aceternity.com/components
   - Animasyonlu, premium React/Tailwind bileşenler
   - **Landing page bu kaynaktan ilham alınarak yapıldı**

4. **Super Design** → https://superdesign.dev
   - Modern SaaS tasarım sistemi örnekleri, AI-assisted tasarım

## Kural

- Yeni UI bileşeni eklemeden önce bu 4 siteden referans al
- Detaylar için: `docs/DESIGN_REFERENCES.md`
